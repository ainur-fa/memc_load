MemcLoad
===================================================

Описание
------------
Cкрипт парсинга и заливки в мемкеш поминутной выгрузкы логов трекера установленных приложений.
* memc_load.py - однопоточная версия
* memc_load_multi.py - более производительный вариант, для конкурентной обработки файла и заливки на сервер используются параллельные процессы.

Пример запуска
------------
```
python memc_load_multi.py --pattern=/Users/s.stupnikov/Coding/otus/*.tsv.gz --dry
```

Совместимость
-------------
Python 3.6 +
